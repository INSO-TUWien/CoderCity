# Code City Backend Service

## Description

Backend service of the code city project. It provides REST-Endpoints for the Code City frontend.
Given a folder it processes all containing git projects and saves the data in a MongoDB database.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
